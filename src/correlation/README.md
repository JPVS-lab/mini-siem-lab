# Correlation Module

Correlaciones que combinan múltiples eventos:
- Reglas de varios pasos.
- Ventanas de tiempo.
- Enriquecimiento básico.

Ejemplos:
- 5 fallos + un login exitoso → "Password Spraying".
- Nuevo usuario + permisos admin en 5 minutos.

Este módulo genera alertas avanzadas para dashboards.
